Healing Hands

Education Portal

Healing Hands is a SvelteKit-based portal for a natural therapy academy with two main roles: students and admins. Students can manage profiles, view enrolled courses, access content, and use a ticket-based support chat. Admins can manage users, courses, applications, enrolments, and support operations.

Platform Features

  • Role-based authentication for student and admin access.
  • Course and enrolment management flows with secure server actions.
  • Support ticket overviews (student + admin), ticket detail chat, and attachment history.

Tech Stack

SvelteKit, Svelte 5 runes, TypeScript, Tailwind CSS v4, DaisyUI, Lucide Svelte, PostgreSQL (Neon), Drizzle ORM + Drizzle Kit, Better Auth, Resend, pnpm, deployed on Vercel. Architecture preference: relational data in Postgres with file metadata in DB and cloud storage for uploads.
